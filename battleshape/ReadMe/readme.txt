 --------------------------------
|BATTLESHAPE ALPHA RELEASE README|
 --------------------------------

Battleshape is an arcade-style 3D space shooter game that pits the player against  waves of deadly geometric shapes intent on destroying planet Earth.


Controls:

WASD & arrow keys: Movement
E: Nuke
P: Pause
+/-: Zoom
Mouse: Aim
Left Click: Shoot

Ship 1 (BST9001):  
This ship has the same control scheme as the classic arcade game 'Asteroids', ie. the ship will accelerate in the direction it is facing.

Ship 2 (UFO):  
This ship has a more literal control scheme as it will accelerate in whatever direction the player chooses.


Instructions:

The player must clear the board of all enemies to progress to the next level using any means at their disposal. Do you have what it takes to complete level 5?
